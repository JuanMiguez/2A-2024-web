Actividad Práctica: CSS Box Model

Objetivo

Aplicar y experimentar con los conceptos del CSS Box Model (content, padding, border, margin) en un archivo HTML proporcionado.

Ejercicio 1: Modificación Individual

Instrucciones:

  1. Descargar y preparar el archivo HTML:

    Descarga y guarda el siguiente archivo HTML como index.html:

    <!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejercicio CSS Box Model</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="box">
        <p>Este es un párrafo dentro de un div con clase "box".</p>
    </div>
</body>
</html>
  2. Crear el archivo CSS:

    Crea un archivo CSS llamado styles.css con el siguiente contenido inicial:

    /* Estilo inicial */
body {
    font-family: Arial, sans-serif;
}

.box {
    width: 300px;
    background-color: #f0f0f0;
}
  3. Modificar el archivo CSS:

    Realiza las siguientes modificaciones en el archivo styles.css:

    Paso 1: Agregar padding a la clase .box

    .box {
    padding: 20px;
}

    Paso 2: Agregar borde a la clase .box

    .box {
    border: 5px solid #333;
}

    Paso 3: Agregar margen a la clase .box

    .box {
    margin: 30px;
}

    Paso 4: Modificar tamaño y color del contenido en la clase .box p

    .box p {
    font-size: 16px;
    color: #333;
}

    Paso 5: Aplicar todas las propiedades a la clase .box y .box p

    .box {
    width: 300px;
    background-color: #f0f0f0;
    padding: 20px;
    border: 5px solid #333;
    margin: 30px;
}

.box p {
    font-size: 16px;
    color: #333;
}

Ejercicio 2: Trabajo en Grupo

Instrucciones:

  1. Descargar y preparar el archivo HTML:

    Descarga y guarda el siguiente archivo HTML como index-group.html:

    <!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejercicio en Grupo - CSS Box Model</title>
    <link rel="stylesheet" href="styles-group.css">
</head>
<body>
    <header>
        <h1>Título del Sitio</h1>
    </header>
    <main>
        <div class="content">
            <p>Contenido principal del sitio web.</p>
        </div>
        <aside class="sidebar">
            <p>Barra lateral con información adicional.</p>
        </aside>
    </main>
    <footer>
        <p>Pie de página</p>
    </footer>
</body>
</html>
  2. Crear el archivo CSS:

    Crea un archivo CSS llamado styles-group.css con el siguiente contenido inicial:

    /* Estilo inicial */
body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 0;
}

header, footer {
    background-color: #333;
    color: #fff;
    text-align: center;
    padding: 10px 0;
}

main {
    padding: 20px;
}

.content {
    background-color: #f9f9f9;
    padding: 10px;
}

.sidebar {
    background-color: #efefef;
    padding: 10px;
}
  3. Modificar el archivo CSS:

    Trabaja en grupo para resolver los siguientes problemas de diseño:

    • Problema 1: Espaciar correctamente el contenido

      Ajusta los márgenes de .content y .sidebar para que haya un espacio de 20px entre ellos.

    • Problema 2: Mejorar la legibilidad del texto

      Añade padding a los elementos de texto dentro de .content y .sidebar.

    • Problema 3: Crear un borde decorativo para el contenido principal

      Añade un borde de 3px, color #ccc y estilo dashed a .content.

    • Problema 4: Alinear el contenido del pie de página

      Añade padding y margen para centrar el texto dentro de footer.

Reflexión y Discusión

Al finalizar los ejercicios, presenta tus soluciones y reflexiona sobre los desafíos encontrados. La discusión debe centrarse en cómo el Box Model afecta el diseño y la funcionalidad de una página web y en cómo se pueden resolver los problemas de diseño de manera colaborativa.

Cierre y Asignación de Tareas

Resumen de la clase: Resaltar los conceptos aprendidos y su aplicación práctica.

Tarea: Diseñar una página web simple que utilice de manera efectiva el Box Model de CSS, aplicando padding, border y margin a distintos elementos.

Estos ejercicios te ayudarán a comprender el CSS Box Model y promoverán el trabajo colaborativo y el desarrollo del pensamiento crítico.
