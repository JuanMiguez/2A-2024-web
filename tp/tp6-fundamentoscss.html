Trabajo Práctico de Interfaz Gráfica Web: Fundamentos de CSS

Objetivo:

Desarrollar habilidades en el uso de CSS para estilizar documentos HTML, enfocándose en la aplicación de selectores, clases e IDs.

Materiales:

  • Computadora con acceso a Internet.
  • Editor de texto: Visual Studio Code.
  • Navegador web.

Instrucciones:

  1. Ejercicio 1: Selector de Elementos
    • Crea un archivo HTML con estructura básica que incluya párrafos, encabezados, listas y enlaces.
    • Utiliza selectores de elementos para aplicar estilos que cambien el color de fondo, el tipo de letra, el tamaño de texto y la alineación.
    • Implementa pseudoclases como :hover y :active para agregar interactividad a los enlaces y botones.
  2. Ejercicio 2: Clases e ID
    • Demuestra y aplica el uso de clases e IDs para asignar estilos específicos a diferentes elementos.
    • Diseña un menú de navegación utilizando clases para su estilización.
    • Crea un formulario de contacto, aplicando estilos diferenciados para cada tipo de input usando IDs.
    • Experimenta con la combinación de selectores para lograr una mayor especificidad y control sobre la aplicación de los estilos.

Entrega:

Guarda tu trabajo en un archivo .html que incluya tanto el HTML como el CSS (puede ser interno o externo). Sube tu archivo al repositorio de GitHub asignado para este curso o a tu perfil personal, en una carpeta específica para este trabajo. Asegúrate de que el repositorio sea público o compartido con el instructor para facilitar la revisión del trabajo.

Criterios de Evaluación:

  • Correcta implementación de selectores, clases e IDs.
  • Creatividad y estética en el diseño.
  • Funcionalidad de las pseudoclases y reactividad de los elementos HTML al interactuar con ellos.
