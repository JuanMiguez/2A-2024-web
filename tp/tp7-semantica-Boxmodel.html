TP7- Box Model y Etiquetas Semánticas

Objetivos del Trabajo Práctico

  • Comprender el funcionamiento del modelo de caja (Box Model) en CSS.
  • Identificar y utilizar correctamente las etiquetas semánticas en HTML5.
  • Desarrollar una página web aplicando los conocimientos adquiridos sobre el Box Model y las etiquetas semánticas.

Instrucciones

Estructura del Documento

Crear un documento HTML5 que incluya las siguientes secciones usando etiquetas semánticas:

  • Header: Con un título principal y una barra de navegación.
  • Main: Contendrá varios artículos dentro de secciones.
  • Aside: Para información adicional o enlaces relacionados.
  • Footer: Con información de contacto y enlaces a redes sociales.

Aplicación del Box Model

Crear un archivo CSS externo donde se apliquen estilos usando el Box Model. Incluir ejemplos de cómo se utiliza padding, margin, y border.

Integración y Validación

Integrar el CSS al HTML vinculando el archivo CSS al documento HTML.

Configuración del Proyecto

Crear una carpeta para el proyecto. Dentro de esta carpeta, crear dos archivos: index.html y styles.css.

Contenido del Archivo index.html

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Trabajo Práctico - Interfaz Gráfica Web</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <h1>Mi Página Web</h1>
        <nav>
            <ul>
                <li><a href="#">Inicio</a></li>
                <li><a href="#">Acerca de</a></li>
                <li><a href="#">Contacto</a></li>
            </ul>
        </nav>
    </header>
    <main>
        <section>
            <article>
                <h2>Artículo 1</h2>
                <p>Contenido del artículo 1...</p>
            </article>
            <article>
                <h2>Artículo 2</h2>
                <p>Contenido del artículo 2...</p>
            </article>
        </section>
    </main>
    <aside>
        <h2>Información Adicional</h2>
        <p>Enlaces relacionados...</p>
    </aside>
    <footer>
        <p>Contacto: email@example.com</p>
        <p>Redes Sociales: Facebook, Twitter, Instagram</p>
    </footer>
</body>
</html>

Contenido del Archivo styles.css

* {
    box-sizing: border-box;
}

body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 0;
}

header, nav, main, aside, footer {
    padding: 20px;
    margin: 10px;
    border: 1px solid #ccc;
}

header {
    background-color: #f8f8f8;
}

nav ul {
    list-style-type: none;
    padding: 0;
}

nav li {
    display: inline;
    margin-right: 10px;
}

main {
    width: 60%;
    float: left;
    margin-right: 2%;
}

aside {
    width: 35%;
    float: left;
}

article {
    margin-bottom: 20px;
}

footer {
    clear: both;
    background-color: #f8f8f8;
}

Evaluación

  • Correcta implementación del Box Model en el diseño.
  • Uso adecuado de etiquetas semánticas.
  • Validez del código HTML y CSS.
  • Responsividad de la página web.
  • Presentación clara y ordenada del código.
